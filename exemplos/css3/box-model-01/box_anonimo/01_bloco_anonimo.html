Box Model bloco Anônimo CSS

Box Nível de bloco Anônimo

A renderização das três linhas de texto se deu como se cada uma delas fosse um box independente.

Isso porque, segundo o Box Model CSS, quando um container possui um elemento nível de bloco,
todos no container se comportam como elementos nível de bloco.

Exemplo de Bloco Anônimo CSS

Se tentarmos colocar uma cor no elemento DIV todos terão a mesma cor.

Exemplo CSS

      div {
         color: blue; /* cor das letras: azuis. */
      }

A regra CSS acima informa para colocar na cor azul todo o texto dentro do elemento Html DIV.

01-Text inserted directly into the div element.

02-This is a paragraph inserted directly into the div element.

03-Text inserted directly into the div element.

No Exemplo acima os textos 01 e 03 são considerados blocos anônimos,
estão soltos no elemento DIV e se comportam como parágrafos.
