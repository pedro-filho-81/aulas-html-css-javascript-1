Bloco Inline Anônimo

Na tag Html DIV tem o elemento inline ( b ) bold e esse elemento inline,
por está no container faz com que todo conteúdo se comporte como elemento
inline e são chamados de boxes anônimos.

Exemplo

      <div>
         Texto no elemento DIV 
         <b>este trecho em negrito</b> 
         mas este trecho normal.
      </div>

Resultado do Exemplo acima

Texto no elemento DIV este trecho em negrito mas este trecho normal.
