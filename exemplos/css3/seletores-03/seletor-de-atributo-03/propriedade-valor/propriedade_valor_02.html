CSS Seletor de Atributo 02

Seletor de atributo é aquele cujo padrão é construído tomando como base um atributo e, seu respectivo valor.

Exemplo CSS

      /* seletor de atributo usando a 
      (propriedade e o valor [title="primeiro"]), 
      desta maneira apenas o parágrafo que está 
      com esse atributo será afetado por essa 
      regra CSS. */
      [title="primeiro"] {
         color: red; /* letras vermelhas */
         font-size: 30px; /* tamanho das letras 30 pixel.*/
      }
