Seletor CSS3 Tag, Propriedade e Valor

Exemplo CSS

p - é a tag, [title] á apropriedade, e ="Terceiro paragrafo" é o valor.

O Exemplo CSS abaixo estiliza o terceiro parágrafo com o fundo amarelo e as letras azuis.

   <style>
      /* Estiliza o terceiro parágrafo */
      p[class|="dest"] {
         width: 500px;
         background-color: yellow;
         color: green;
         font-size: 25px;
      }
   </style>

No exemplo mostrado, os textos do primeiro, terceiro e quinto parágrafos serão na cor verde (green), conforme definido na regra CSS.
