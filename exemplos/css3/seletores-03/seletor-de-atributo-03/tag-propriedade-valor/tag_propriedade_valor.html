Seletor CSS3 Tag, Propriedade e Valor

Exemplo CSS

p - é a tag, [title] á apropriedade, e ="Terceiro paragrafo" é o valor.

O Exemplo CSS abaixo estiliza o terceiro parágrafo com o fundo amarelo e as letras azuis.

   <style>
      /* Estiliza o terceiro parágrafo */
      p[title="Terceiro paragrafo"] {
         background-color: yellow;
         color: blue;
         font-size: 25px;
      }
   </style>
