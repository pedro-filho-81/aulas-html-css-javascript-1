CSS Seletor de Atributo

Seletor de atributo é aquele cujo padrão é construído tomando como base um atributo e, seu respectivo valor.

Exemplo CSS

      /* seletor de atributo usando apenas a 
      (propriedade [title]), desta maneira 
      todos os parágrafos com essa propriedade 
      são afetadas por essa regra CSS3.*/
      [title] {
         color: red;
      }
