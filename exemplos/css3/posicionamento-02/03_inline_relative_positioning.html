Inline Relative Positioning CSS

As propriedades left e right definem o quanto o bloco deve ser deslocado para a direita ou esquerda, respectivamente, e as propriedades top e bottom o deslocamento para baixo ou para cima,respectivamente.

Elementos inline também podem ser deslocados da sua posição padrão com uso dessa declaração.

Paráragrafo original

Nesse texto esta palavras ficara em negrito e será deslocada.

Exemplo css

      /* regras css para o elemento inline strong */
      #resultado strong {
         background: yellow; /* fundo amarelo */
         color: green;       /* letras verdes */
         
         position: relative; /* posição relativa preserva a posição inicial */
         left: 60px;         /* move da esquerda para a direita 60 pixel  */
         bottom: 20px;       /* move 20 pixel para cima */
      }
