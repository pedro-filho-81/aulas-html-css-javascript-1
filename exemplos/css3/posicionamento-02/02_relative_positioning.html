Relative Positioning CSS

No Position relative a posição anterior é preservada.

As propriedades left e right definem o quanto o bloco deve ser deslocado para a direita ou esquerda, respectivamente, e as propriedades top e bottom o deslocamento para baixo ou para cima,respectivamente.

Exemplo Css

   .two {
      width: 300px;
      border: 2px solid black;
      background: blue;
      color: white;

      position: relative;
      left: 200px; /* Desloca 200 pixel da       
                     esquerda para direita */
      top: 20px; /* Desloca 20 pixel da posição 
                     de origem para baixo. */
   }
